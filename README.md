# Java-UUID-Generator
可以一次產生多個uuid
## 下載
<a href="https://github.com/ben99933/Java-UUID-Generator/releases">載點</a>
## 用法
在命令列使用以下語法
>java -jar uuid.jar
<p>
  當你想要一次產生多個uuid時可以在後面加上數字
</p>

>java -jar uuid.jar **amount**

<p>
 參數amount的最大值可以到32767<br>
  或者你可以直接使用max來達到最大值
</p>

>java -jar uuid.jar max
